##Spring Boot 프로젝트
이 프로젝트는 Spring Boot를 사용하여 개발되었습니다.

###버전 정보
Spring Boot: 2.4.2
Spring Dependency Management: 1.0.15.RELEASE
사용된 라이브러리
Spring Boot Starter Web
MySQL Connector Java
MyBatis Spring Boot Starter
Log4jdbc-log4j2-jdbc
Gson
Lombok

###사용 방법
1. 이 프로젝트를 클론합니다.
2. 필요한 의존성을 설치합니다.
3. 애플리케이션을 실행합니다.
